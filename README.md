MarkBin is a what-you-see-is-what-you-get (WYSIWYG) Markdown editor that supports standard Markdown syntax and GitHub Flavored Markdown.

It supports both source code editing and WYSIWYG modes with free switching between them, code highlighting, and table insertion with row and column sorting, moving, and other operations. It also allows inserting images, hyperlinks, mathematical formulas, and more.
The software supports internationalization with 45 languages, customizable UI themes and code block themes, automatic backup, as well as a left-side table of contents navigation bar.
